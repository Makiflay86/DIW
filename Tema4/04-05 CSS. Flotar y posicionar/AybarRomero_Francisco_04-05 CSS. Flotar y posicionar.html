Ejercicio 1

Float: None

Este texto se mantiene debajo de la imagen porque no tiene ninguna propiedad de desplazamiento lateral aplicada.

Float: Left

Al usar float left, la imagen se pega a la izquierda y este texto sube para rodearla por el espacio disponible a la derecha.

Float: Right

En este caso, la imagen se desplaza al extremo derecho y el flujo del texto se ajusta para aparecer en el lado izquierdo.

Ejercicio 2

Clear: None

Este texto tiene clear: none, por lo que fluye al lado de la imagen aprovechando el espacio vacío.

Clear: Left

Este texto tiene clear: left, por lo que ignora el espacio lateral y baja hasta limpiar la imagen flotada a la izquierda.

Clear: Right

Este texto tiene clear: right. Como la imagen flota a la derecha, el texto baja para empezar debajo de ella.

Clear: Both

Este texto tiene clear: both, lo que asegura que se posicione debajo de ambas imágenes, sin importar sus flotaciones.
